# creation of case

URL for registration 
localhost:1978/cases/registerCase

**Paramaters are passed by the value**

```
"victim" : _string_
"fir" : _string_
"policeStation" : _string_
"areaPincode" : _Number_
"IPCs" : _ array of strings_
"prevCaseId" : _string_
```

_example :_

>{
>       "victim": "Saquib",
>        "fir": "529810-123",
>        "policeStation": "Chutiya",
>        "areaPincode": 480001,
>        "IPCs": ["295","296","297"],
>        "prevCaseId": "4567-123526"
>}
